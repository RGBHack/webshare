## 🛠️ Usage

Bors is extremely easy to set up and host:

```sh
# 1. clone the repository
git clone https://github.com/safinsingh/bors

# 2. set up token
echo "TOKEN=MY_BOT_TOKEN" >.env

# 3. build for release
cargo build --release -vv
cp target/release/bors .

# 4. run!
LOG= ./bors
```
